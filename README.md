# Power BI Sales Performance Dashboard

## Objective
Design an executive-level sales dashboard to monitor performance, trends, and key business drivers using Power BI.

## Dataset
Public sales dataset containing orders, customers, products, regions, and financial metrics.

## Key KPIs
- Total Sales
- Total Orders
- Total Profit
- Profit Margin

## Features
- Interactive filters by date, region, and sub category
- Trend analysis over time
- Top-performing sub category and customers
- Clean star-schema data model

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Data Modeling
- CSV data source

## Business Value
This dashboard enables leadership teams to quickly identify sales trends, regional performance gaps, and growth opportunities.
